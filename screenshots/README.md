# Screenshots Directory

This directory contains screenshots of the Orbital CLI application in action. Below are the recommended screenshots to capture:

## Recommended Screenshots

1. **CLI Login Process**
   - Screenshot of the terminal when running `orbit login`
   - Show the device code generation
   - Include the verification URL

2. **Device Authorization Web Interface**
   - Screenshot of the device authorization page in the browser
   - Show the input field for the device code

3. **Device Approval Screen**
   - Screenshot of the device approval page
   - Show the user information and approval options

4. **CLI Chat Interface**
   - Screenshot of the AI chat in the terminal
   - Show a conversation with the AI assistant

5. **Web Dashboard**
   - Screenshot of the main dashboard page
   - Show user profile information

## Adding Screenshots

To add screenshots:

1. Capture the required screenshots as listed above
2. Save them in this directory with descriptive names:
   - `cli-login.png`
   - `device-auth.png`
   - `device-approval.png`
   - `cli-chat.png`
   - `web-dashboard.png`
3. Update the main README.md file to uncomment the image references

## Image Requirements

- Format: PNG preferred
- Size: Optimize for web (under 1MB each)
- Content: Clear and readable, showing the key features
- Annotations: Consider adding annotations to highlight important elements
